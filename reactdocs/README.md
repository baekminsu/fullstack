# React + JavaScript 상세 학습 문서

이 폴더는 JavaScript 기본 문법부터 React 애플리케이션 설계까지 이어지는 세부 학습 문서입니다. React를 제대로 쓰려면 JavaScript의 값, 함수, 객체, 비동기, 브라우저 동작을 먼저 이해해야 합니다.

## 읽는 순서

1. [01-javascript-foundations.md](./01-javascript-foundations.md): JavaScript 실행, 변수, 타입, 연산자
2. [02-javascript-functions-objects-async.md](./02-javascript-functions-objects-async.md): 함수, 객체, 배열, 비동기
3. [03-browser-dom-events-and-web-api.md](./03-browser-dom-events-and-web-api.md): DOM(Document Object Model), 이벤트, fetch, 브라우저 API(Application Programming Interface)
4. [04-react-foundations-jsx-components.md](./04-react-foundations-jsx-components.md): React 기본, JSX, 컴포넌트
5. [05-react-state-effects-forms.md](./05-react-state-effects-forms.md): state, effect, form
6. [06-react-routing-server-state.md](./06-react-routing-server-state.md): 라우팅, 서버 상태, API(Application Programming Interface) 연동
7. [07-react-performance-testing-accessibility.md](./07-react-performance-testing-accessibility.md): 성능, 테스트, 접근성
8. [08-react-project-curriculum.md](./08-react-project-curriculum.md): 단계별 프로젝트

## 학습 원칙

- 화면이 어떻게 보이는지보다 상태가 어떻게 흐르는지를 먼저 봅니다.
- 로딩, 에러, 빈 상태를 기본 기능의 일부로 다룹니다.
- JavaScript의 비동기와 참조 타입을 이해하지 않은 상태로 React 최적화를 하지 않습니다.
